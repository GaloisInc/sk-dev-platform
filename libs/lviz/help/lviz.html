Overview

Lobster DSL (Domain Specific Language) is a high level policy specification language which provides methods of abstracting, understanding, and analyzing security policies.  LViz provides a method of visualizing the policy in a way that improves understanding.

Invoking LViz

LViz can be started through the lobster command line via the command:
lviz [optional lobster file]
If the optional file is used, it will be loaded at startup. If not provided, a file can be loaded through the "file/open" menu item.
Once lviz is invoked, it provides a normal GUI interface with menu options.  Menu choices are listed below.

The initial version of lviz is a viewing only tool.  Later version may be exampled to allow editing and to simplify Lobster policy creation.

Mouse

The mouse cursor currently does not support selection of screen items.  This functionality will be added to a later version of lviz.  The main existing selection mechanism is the ability to expand and collapse the display of domain hierarchy.  For some Lobster policies, seeing only the top level can aid in understanding the information flow.  To expand collapse a domain - put the cursor over the domain rectangle and right click the mouse. The two choices are expand and collapse that individual domain. This differs from the menu selection which affects all domains.
